自訂函數的輸入與輸出個數¶

單一輸入¶

將攝氏溫度轉換為華氏溫度的函數 celsius_to_fahrenheit()

\begin{equation} Fahrenheit_{(°F)} = Celsius_{(°C)} \times \frac{9}{5} + 32 \end{equation}
In [29]:
celsius_to_fahrenheit <- function(x) {
    return(x*9/5 + 32)
}
print(celsius_to_fahrenheit(100))

[1] 212

兩個以上輸入¶

In [30]:
get_bmi <- function(height, weight) {
    return(weight / (height*0.01)**2)
}
print(get_bmi(216, 147))

[1] 31.5072

參數有預設值¶

In [31]:
temperature_convert <- function(x, to_fahrenheit = TRUE) {
    if (to_fahrenheit) {
        return(x*9/5 + 32)
    } else {
        return((x - 32)*5/9)
    }
}
print(temperature_convert(100))
print(temperature_convert(212, to_fahrenheit = FALSE))

[1] 212
[1] 100

多個輸出¶

In [32]:
get_bmi_and_label <- function(height, weight) {
    bmi <- weight / (height/100)**2
    if (bmi > 30) {
        label <- "Obese"
    } else if (bmi < 18.5) {
        label <- "Underweight"
    } else if (bmi > 25) {
        label <- "Overweight"
    } else {
        label <- "Normal weight"
    }
    bmi_and_label <- list(
        bmi = bmi,
        bmi_label = label
    )
    return(bmi_and_label)
}
In [33]:

[1] 31.5072

$bmi_label
[1] "Obese"

$bmi
[1] 27.42119

$bmi_label
[1] "Overweight"

向量化函數¶

什麼是向量化函數？¶

利用函數型函數（Functional functions）將指定函數向量化至某個資料結構的技巧。

In [34]:
# 將一個 list 中的每個數字都平方
my_list <- list(
    11,
    12,
    13,
    14,
    15
)
print(my_list)

[[1]]
[1] 11

[[2]]
[1] 12

[[3]]
[1] 13

[[4]]
[1] 14

[[5]]
[1] 15

老實說，我們會有個衝動這樣寫¶

my_list**2

注意這是一個 list 而不是向量¶

In [35]:
for (i in my_list) {
    print(i**2)
}

[1] 121
[1] 144
[1] 169
[1] 196
[1] 225

使用 lapply() 將 get_squared() 函數向量化至 my_list 之上¶

In [36]:
get_squared <- function(x) {
    return(x**2)
}

print(lapply(my_list, FUN = get_squared))

[[1]]
[1] 121

[[2]]
[1] 144

[[3]]
[1] 169

[[4]]
[1] 196

[[5]]
[1] 225

搭配匿名函數（Anonymous functions）將平方運算向量化至 my_list 之上¶

# Named function
named_function <- function(INPUTS) {
    # sequence of statements
    # sequence of statements
    return(OUTPUTS)
}
# Anonymous function
function(INPUTS) return(OUTPUTS)
